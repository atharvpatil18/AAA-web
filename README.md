## 🌟 Overview

Arnav Abacus Academy offers premium offline Abacus and Vedic Maths coaching under the direct guidance of certified Master Trainer **Neha Patil**. 
With over 200+ local families enrolled, our classes utilize sensory bead manipulation and photographic mental calculations to eliminate math phobia, develop split-second concentration, and build deep math confidence for students aged 4 to 16.

This web application serves as the primary academy center portal, offering parents full curriculum visibility, counselor advice slots, address directions, verified parent reviews, and a interactive Diagnostic Math Challenge widget.

---

## 🎯 Key Features

* **Speed Challenge Widget**: An interactive 30-second mental math test designed for kids of different age ranges (4-6, 7-9, 10+) to test their speed, with direct links to book corresponding free physical diagnostic evaluations.
* **Stream-focused Programs**: Clean showcases describing age-specific pathways (Abacus streams, Vedic math benchmarks, and regular School board readiness checks).
* **Direct Lead Generation**: Dynamic inquiry form coupled with analytics and pre-filled WhatsApp direct chat integrations to coordinate child trail seats.
* **Verified Testimonials Carousel**: Genuine, scannable feedback from Wakad parents reflecting abacus center outcomes.
* **Interactive Directions**: Integrated Google Maps iframe displaying the precise Wakad address alongside operational class timings.
* **Modern Interface**: A beautiful theme utilizing generous negative space, vibrant warm accent templates, fluid transitions, and a solid responsive structure.

## 🎓 Curriculum & Programs

### 1. Beginner Abacus Stream (Ages 4-6)
Focuses on tactile bead movements, single digit additions, and spatial memory indexing to promote left-and-right brain synchronization.

### 2. Intermediate Abacus Stream (Ages 7-9)
Transitions from physical abacus frames into mental abacus cards. Covers multi-row addition & subtraction chains without paper drafts.

### 3. Advanced Vedic Maths & School Prep (Ages 10+)
Empowers older children to execute double/triple digit cross-multiplications, structural algebra, squares, and division shortcuts up to 10-15x faster than standard calculations.

## 🚀 Getting Started (Local Development)

To run this application locally, ensure you have **Node.js** (v18 or higher) installed on your system.

### 1. Clone the Workspace
Retrieve or export the source files to your local system.

### 2. Install Dependencies
Execute the standard package manager installer inside the root folder:
```bash
npm install
```

### 3. Environment Setup
Create a `.env` file in your root directory based on the `.env.example` file:
```bash
cp .env.example .env
```

### 4. Direct Development Server
To boot the hot-reloading local web dev-server:
```bash
npm run dev
```
The application will run on [http://localhost:3000](http://localhost:3000) by default.

### 5. Production Build
Prepare optimised static compilation files inside the `/dist` directory:
```bash
npm run build
```
